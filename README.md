# EDA with Excel
Exploratory Data Analysis (EDA) of Titanic Passengers Using Microsoft Excel
This project explores the Titanic dataset to uncover insights about passenger survival rates using Microsoft Excel. The analysis includes:

Data Cleaning & Preparation – Handling missing values and formatting data for better analysis.
Descriptive Statistics – Using Data Analysis ToolPak to calculate survival rates, passenger demographics, and ticket class distributions.
Data Visualization – Creating histograms, boxplots, and bar charts to analyze survival by age, gender, and class.
Pivot Tables & Charts – Summarizing survival patterns and visualizing key trends interactively.
This EDA provides valuable insights into factors affecting survival on the Titanic, helping to understand data trends in a business analytics context.
